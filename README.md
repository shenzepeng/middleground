Business Capability
Saas平台

1.一件生成商家小程序
  1.1 商家通过一件生成小程序进行商品管理
  1.2 商家绑定自身的收款账号，用户在小程序下单
  1.3 商家通过小程序完成这个商品的营销

2.技术能力
  2.1文件管理
  2.2图片上传
  2.3权限管理
  2.4智能检索
  2.5图像识别，客户通过自己想要商品图片搜索到类似的商品
